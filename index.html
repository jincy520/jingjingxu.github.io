Software

A Resource-independent Word Segmentation Toolkit

This toolkit is a word segmentation framework on top of c# that aims to provide an efficient way of implementing deep neural networks.

Main features:

  • This tooklit is developed with C#. Traditional neural networks are usually developed with Python. We supply the version of C# to help developers build word segmentation systems.
  • This toolkit achieves competitive results on MSR, PKU and CTB (8.0) datasets.
  • To speed up training, we explore mini-batch asynchronous parallel learning on neural word segmentation.
